# Statistical Analysis Examples in Python

This repository demonstrates real-world statistical methods applied to business/product analytics using Python.  
It includes:

1. **Descriptive Statistics** – Summarizing user behavior data.
2. **A/B Testing** – Hypothesis testing on feature impact.
3. **Correlation Analysis** – Understanding relationships between metrics.

## Setup

```bash
git clone https://github.com/yourusername/statistical-analysis-examples.git
cd statistical-analysis-examples
pip install -r requirements.txt
```

Folder Structure

- notebooks/ – Jupyter notebooks for interactive analysis
- scripts/ – Python scripts for each statistical method
- data/ – Optional datasets

Example Usage

```bash
python scripts/descriptive_stats.py
python scripts/ab_testing.py
python scripts/correlation_analysis.py
```
