# Data Analysis Projects

The projects demonstrate data exploration, analysis, and visualization using tools commonly used in data analytics workflows.

## Tools and Libraries

- Python  
- Pandas  
- NumPy  
- Seaborn  
- Plotly  
- Streamlit

## Projects

### 1. Ecommerce Sales Analysis
Exploratory data analysis of ecommerce sales data to identify revenue trends, product performance, and seasonal patterns.

**Techniques used**
- Data cleaning with Pandas  
- Aggregations and group-by analysis  
- Visualization with Seaborn and Plotly  

### 2. Customer Churn Analysis
Analysis of customer behavior to understand churn patterns and key factors influencing retention.

**Techniques used**
- Exploratory data analysis  
- Feature analysis  
- Visualization of churn patterns  

### 3. Financial Time Series Analysis
Analysis of financial time series data to study trends, volatility, and correlations.

**Techniques used**
- Time series analysis  
- Statistical visualization  
- Interactive plots with Plotly  

### 4. Marketing Campaign Analysis
Analysis of marketing campaign data to evaluate effectiveness and ROI.

**Techniques used**
- Segmentation analysis  
- Conversion rate analysis  
- Visualization of campaign performance  

### 5. Streamlit Dashboards
Interactive dashboards created using Streamlit for exploring datasets dynamically.

**Examples include**
- Sales dashboard  
- Stock dashboard  

### 6. Interactive Plotly Analysis
Interactive data visualizations using Plotly to explore trends, correlations, and patterns.

**Techniques used**
- Line, bar, scatter, and heatmap charts  
- Interactive hover and filter functionality  
- Time-series and categorical analysis
